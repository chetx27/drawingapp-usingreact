# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-02

### Added
- Professional README with comprehensive documentation
- CI/CD pipeline with GitHub Actions
- Package.json with complete project configuration
- TypeScript support throughout the codebase
- Code of Conduct and Contributing guidelines
- MIT License
- Changelog for version tracking
- Issue and PR templates
- Comprehensive testing setup (Jest, Cypress)
- ESLint and Prettier configuration
- Project architecture documentation

### Changed
- Restructured project for better maintainability
- Enhanced UI/UX with modern design patterns
- Improved performance with optimized rendering
- Updated dependencies to latest stable versions

### Fixed
- Canvas rendering performance issues
- Memory leaks in event listeners
- Mobile touch event handling
- Color picker responsiveness

## [1.0.0] - 2025-12-01

### Added
- Initial release with basic drawing functionality
- Freehand drawing tool
- Basic shape tools (rectangle, circle, line)
- Color picker
- Clear canvas functionality
- Export as PNG
